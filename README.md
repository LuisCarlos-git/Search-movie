# Search movies

![gif](./movie.gif)

## Sobre

Uma aplicação simples para pesquisa de filmes, ver os mais votados na pagina inicial, ver os detalhes do filme como nome, sinopse, e trailer

## Tecnologias

- react
- axios
- styled-components
- react-icons
- eslint
- prettier

## Como iniciar ?

rodar o comando abaixo para instalar todas as dependencias

```shell
yarn

```
